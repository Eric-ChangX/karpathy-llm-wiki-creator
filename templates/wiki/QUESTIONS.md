# Open Questions

待解的问题队列。LLM 在 ingest / query 中遇到的悬而未决问题来这里登记，定期用新来源回访。

每条格式：

```
## Q-NNN: 一句话问题
- **状态**: open | answered | dropped
- **提出**: YYYY-MM-DD（来源 [[YYYY-MM-DD_短标题]]）
- **关联**: [[相关 entity / concept / synthesis 页]]
- **背景**: 1-3 句话说明为什么是问题
- **当前最佳猜测**: 如有
- **回答时机**: 什么样的新来源/事实能 close 它
```

answered 状态的问题不立即删，保留 1-2 个 ingest 周期作为 trail，再清理。
